
What to do with this file ?

1. Download Immunity Debugger (see my links page or try Google)
2. Install it ..
3. Open the directory it is installed to ..
4. Copy the Python file into the directory called PyCommands ..
5. Open Immunity Debugger, Press F5 to load PyCommands window ..
6. In the bar at the bottom of the main window type "!usage bpxep"
7. Usage instructions are written to the log window ..
8. Have fun!
